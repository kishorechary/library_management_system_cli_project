# library_management_system_cli_project
# 📚 Library Management System (Python CLI)

A simple command-line based Library Management System built using Python. This project allows users to manage books efficiently by performing operations like adding, searching, issuing, and deleting books.

---

## 🚀 Features

* Add Book
* Show All Books
* Search Book
* Delete Book
* Issue Book
* Update Book Status

---

## 🛠️ Technologies Used

* Python
* Lists & Dictionaries

---

## ▶️ How to Run

1. Clone the repository or download the files
2. Open terminal / command prompt
3. Run the program:

```id="k9b3c2"
python library_management.py
```

4. Select options from the menu
